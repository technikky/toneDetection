## 1. What this software is

Sight-Singing Studio is a music-education application that runs entirely on a
school's local computer. It does not require or use an internet connection to
operate, and does not transmit data to `[COMPANY NAME]` or anyone else over
the network at any point during normal use.

## 2. What we collect, and from whom

When a teacher uses Sight-Singing Studio with their class, the software stores
the following **locally, on that classroom computer only**:

- **Teacher account information**: a username and a securely hashed password
  (the plain password is never stored).
- **Student roster entries**: a student's name and an internal "consent on
  file" flag, entered by the teacher. Names are encrypted at rest.
- **Access codes**: a short code generated per student, used in place of a
  name or login, so a student does not need to create an account.
- **Audio recordings**: short clips of a student singing an assigned
  exercise, captured when the student submits a take for grading.
- **Assessment scores**: pitch-accuracy and pronunciation-accuracy scores,
  and a per-note breakdown, generated by analyzing each recording.

We do not collect a student's date of birth, address, email, or any
information beyond what's listed above. We do not collect information from
students directly -- the teacher enters roster information on the student's
behalf.

## 3. Where this data goes

**Nowhere, by default.** All of the above is stored in a local database file
and local audio files on the computer running the application. Sight-Singing
Studio does not upload this data to `[COMPANY NAME]`, to any cloud service, or
to any third party. There is no analytics or telemetry collection in the
current version of this software.

Data is kept locally for as long as the school/teacher chooses to keep it.
There is currently no automatic deletion schedule -- **the school is
responsible for deciding and implementing a retention policy** (e.g.,
deleting recordings and roster entries at the end of each school year).

## 5. We do not sell your data

We do not sell, rent, or trade student data, recordings, or scores to any
third party for any purpose, including advertising.

## 6. Parental / guardian consent

Because this software records children's voices, we require a signed
consent form (see `PARENT_CONSENT_FORM_TEMPLATE.html` in this same folder)
before a student is recorded. It is the school's/teacher's responsibility to
obtain and keep this consent on file -- the "consent on file" flag in the
roster is a self-reported record of that, not independent verification.

- Teacher accounts are protected by a locally-hashed password (scrypt); we
  never store or transmit plaintext passwords.
